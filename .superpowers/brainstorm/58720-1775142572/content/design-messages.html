What changed

  • Square avatars — rounded squares (border-radius 4px) instead of circles. More GitHub-native.
  • Edit user messages — ✏ Edit button on hover replaces old user bubble inline. Clicking opens the bubble in an editable state and re-sends.
  • Timestamps + generation time — subtle timestamp on each message; assistant messages also show generation duration (e.g. "0.8s").
  • Code blocks with line numbers — gutter with line numbers, colored language dot in header, cleaner copy button.
  • Streaming indicator — during streaming, a pulsing "Generating…" badge appears below the last assistant bubble instead of the blinking cursor (more legible).
  • Bordered assistant bubble — adds a subtle #30363d border to assistant messages for better separation on dark backgrounds.

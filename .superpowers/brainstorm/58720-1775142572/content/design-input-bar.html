What changed

  • Slash commands — type / to trigger a filtered popup (e.g. /explain, /fix, /clear). Extensible list.
  • System prompt pill — collapsed toggle above the box. Click to expand an inline editor. Shows active preset name.
  • Token counter — live estimate in the toolbar. Turns amber near the model's context limit.
  • Keyboard hint strip — ↵ Send · ⇧↵ Newline, always visible. Replaces the "AI can make mistakes" disclaimer (move to settings).
  • ⌘K hint — passive reminder of the command palette shortcut (for future use or opt-in).
